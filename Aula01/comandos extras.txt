cmd comands:

    ls - list (listar pasta atual)
    rm - remove (remover arquivo)
    rmdir - remove pastas
    mv - rename (renomeia arquivo)
    mvdir - renomear ou mover pastas
    mkdir - make diretory (criar diretório ou pasta)
    mv <arquivo> <diretório>/ - Move arquivo para diretório
    mv <arquivo> .. - move o arquivo a um nível acima na arvore. obs.: creio que isso deve funcionar junto com ../.. para subir dois níveis
    cd - Change directory - (muda o diretório)
    cp - copy - (copiar)
    make <script>.c - copilador será executado para o arquivo desejado
